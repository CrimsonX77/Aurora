## Security Policy

If you discover a security issue in this project, please contact the repository owner privately at bjornwalczak@gmail.com. Do not open an issue containing sensitive details.

This repository does not contain production secrets. If you find credentials, API keys, or other secrets, please report them privately so they can be revoked and removed from the repository history.

Guidance:
- Do not commit secrets or `.env` files.
- Add runtime secrets to environment variables or a secret manager.
- Rotate keys immediately if exposed.
