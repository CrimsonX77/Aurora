# Aurora

Sanitized public README for the Aurora project.

Overview
- Aurora provides tools for card generation, steganography-based embedding, and member management. The full project includes both research and operational code. Sensitive pieces such as cryptographic keys, local `.env` files, and test fixtures are excluded from this public repository.

Quickstart

1. Create and activate a virtualenv:

```bash
python -m venv venv
source venv/bin/activate
```

2. Install requirements:

```bash
pip install -r requirements.txt
```

3. Configure runtime secrets via environment variables or a secrets manager. See `SECURITY.md`.
